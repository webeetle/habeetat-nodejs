title: Installare Node.js
---

## Utenti windows

Se sei un utente Windows, ti consiglio vivamente di [installare l'ultima versione](https://nodejs.org/en/) LTS dal sito web ufficiale di Node.js 


## Per tutti gli altri

Se non utilizzi il sistema operativo Windows, ti consiglio vivamente di installare [`nvm` (node version manager)](https://github.com/nvm-sh/nvm).
NVM ti consente di installare tante versioni di Node.js sul tuo sistema con la possibilità di passare da una versione all'altra con estrema semplicità.
Inoltre, NVM installa Node.js in una cartella che non avrà errori di autorizzazione che altrimenti incontreresti con il programma di installazione ufficiale.

Dopo aver installato nvm, è necessario installare una versione Node. Puoi scaricare l'ultima versione LTS con questo comando.


`nvm install --lts`
